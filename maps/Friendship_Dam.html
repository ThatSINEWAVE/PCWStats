Friendship Dam

The main gameplay idea behind this map was to create a map with a large elevation difference so the map is divided into two main height levels (upper and lower). The team wanted to create a map that requires being aware of your surroundings not just horizontally, but also vertically. The base layout for Conquest is a kite formation. Currently, this map supports the Conquest and Domination game modes.

Map Features

1 home base for each team

3 equivalent bases

2 Team spawn points
